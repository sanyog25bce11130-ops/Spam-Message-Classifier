# Spam Message Classifier

This is a simple machine learning project where I built a model to classify messages as **spam** or **not spam (ham)**.

## 📌 About the Project
Spam messages are very common these days (offers, scams, ads, etc.).  
The goal of this project is to automatically detect whether a message is spam or not using basic machine learning techniques.

## 🧠 What I Used
- Python  
- Pandas (for data handling)  
- Scikit-learn (for ML model)  
- Matplotlib (for visualization)  

## 📂 Dataset
I used a small dataset of messages labeled as spam or ham for training the model.

## ⚙️ How it Works
1. Load the dataset  
2. Convert text into numerical form  
3. Train a machine learning model  
4. Predict whether a message is spam or not
